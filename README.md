# Pharmaceutical Demand Forecasting Dashboard

## Overview
This project predicts monthly pharmaceutical sales using lag-based time-series forecasting and presents results in an interactive Dash dashboard.

## Features
- Lag-based Machine Learning model
- R² and MAE evaluation
- Growth simulation slider
- KPI summary cards
- Interactive category selection

## Tech Stack
- Python
- Scikit-learn
- Dash (Plotly)
- Pandas & NumPy

## Model Performance
R²: 0.19  
MAE: 340 units
